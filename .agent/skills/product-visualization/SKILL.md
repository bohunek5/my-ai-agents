# Product Visualization (B.L.A.S.T. Style)

## When to Use

- Creating premium hero images.
- Generating end-frame reveal shots (exploded views).
- Defining transition flows for scrollytelling.

## Prompt Templates

### 1. Hero / Start Frame

- Floating centered, slight natural tilt.
- Studio lighting, pure solid background (`#050505`).
- Sharp branding, premium gloss.

### 2. Exploded / End Frame

- "Apple-style" industrial engineering render.
- Components separated along axis, suspended in mid-air.
- Hyper-real materials (metal, glass, plastic).

### 3. Transition Flow

- Assembled to Exploded motion.
- Elements entering from behind, building momentum.
- No motion blur, ultra-sharp focus.

## Instructions

- Always use WebP for final project integration.
- Background MUST match exactly: `#050505` for seamless scrollytelling.
