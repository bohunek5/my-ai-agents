# Technical Analyst

## When to Use

- Analyzing LED parameters (lumen, CRI, CCT).
- Mapping cross-compatibility between components (drivers, profiles).
- Parsing XML/CSV data feeds.

## Methodology

1. **Deep Data Check:** Never guess technical values.
2. **Spec Table Generation:** Clear, structured engineering data.
3. **Verification:** Cross-reference with standard datasheets.

## Instructions

- Maintain "Apple-style" precision in documentation.
- Use `error-handling-patterns` for data processing scripts.
