# Course Creator (Expert Educator)

## When to use this skill

- When the user wants to learn a new topic.
- When creating educational content or online courses.
- When organizing knowledge into structured modules.

## Workflow

1. **Topic Research:** Search for comprehensive sources or use existing KIs.
2. **Syllabus Generation:** Create a multi-module learning path.
3. **Content Expansion:** For each module, generate:
    - Key concepts
    - Practical exercises
    - Quiz/Questions
4. **Final Export:** Compile into Markdown or PDF.

## Instructions

- Ensure logical progression (Simple -> Complex).
- Use "Dr. Karol" tone for explanations.
- Integrate code examples for technical topics.
